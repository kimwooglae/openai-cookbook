
스튜디오 설치3.스튜디오 설치3.1인스톨러 설치3.1.1설치 전 준비(1) 인스톨러 준비사용자 PC 사양에 맞는 인스톨러를 제공 받으십시오. (2) Java 1.8 준비WebSquare Studio IWD는 Java 1.8 이상에서 동작합니다. 사용자 PC에 Java 1.8 이상을 설치하십시오. (3) 기존 스튜디오 삭제기존에 설치된 스튜디오가 있을 경우, 설치된 스튜디오를 먼저 제거하십시오.단, workspace 폴더는 삭제하지 말고 계속 사용하십시오. (4) 32-bit (x86) 인스톨러 사용64-bit OS에서 32-bit JRE를 사용하려면, 32-bit (x86) 인스톨러를 실행하십시오.3.1.2인스톨러 설치인스톨러를 실행합니다.다음을 클릭합니다.동의함을 클릭합니다.WebSquare Studio IWD는Java 1.8 이상에서 동작합니다. Java 경로를 bin 폴더까지 지정한 후 다음을 클릭합니다.설치 경로를 지정합니다. (기본 경로: C:\WebSquare_Studio\x86\websquare)아래와 같이 설치가 진행됩니다.다음을 클릭합니다.성공적으로 설치되면 아래와 같은 화면이 나타납니다.Studio를 실행하십시오.Workspace를 지정합니다.Studio가 시작되고 아래와 같이 웰컴 페이지가 표시됩니다. 각 항목을 클릭하면 해당 내용을 확인할 수 있습니다.동영상스튜디오 사용법 설명 동영상 모음Help – 웹스퀘어 라이선스 등록을 선택하여, 라이선스를 입력합니다.라이선스가 등록되면 아래와같은 창이 나타납니다.3.2Eclipse 플러그인 설치3.2.1설치 전 준비WebSquare5 Studio를 설치하려는 PC에 JDK(Java Develoment Kit) v1.8 버전이 설치되어 있어야 합니다.한글 포함 금지 및 글자수 제한설치 폴더명이 한글을 포함하거나 256자를 초과할 경우 WebSquare5 Studio가 정상적으로 작동하지 않을 수 있습니다.제약 사항(1) Window OS 만 지원.(2) 최소 JRE(JDK) 1.8.0_2xx 설치가 필요. .(3) openjdk-1.8.0.242-1.b08.x86_64 : 자체 버그로 인해 이클립스 프로젝트가 생성되지 않고 스튜디오가 비정상 동작. 다른 버전 사용 권장. (4) 기존 웹스퀘어 스튜디오 버전에서 제공되던 jre 포함 버전은 미지원.(5) Java SE 8 상위 버전은 지원 범위 제외.3.2.2Eclipse 플러그인 설치WebSquare-feature 파일을 준비합니다.

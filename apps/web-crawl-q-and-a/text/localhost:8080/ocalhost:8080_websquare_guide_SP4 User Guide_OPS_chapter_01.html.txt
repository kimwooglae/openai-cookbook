
구성1.구성WebSquare5는 Engine과 개발도구(WebSquare5 Studio)로 구성됩니다. Engine은 클라이언트 요청을 받아 WebSquare5 화면 파일을 브라우저에 표시합니다. 개발자는 WebSquare5 Studio에서 화면 파일을 개발할 수 있습니다.1.1StudioWebSquare5 Studio는 WYSIWYG 방식의 통합 개발 도구로, 개발자는 컴포넌트 그리기, 스크립트 추가, 화면 확인, 디버깅을 한번에 수행할 수 있습니다. 본 문서는 WebSquare5 Studio IWD(Intelligent Web Designer) 버전을 기준으로 사용법을 설명합니다.1.1.1제공 형태표 1-1.WebSquare5 Studio (IWD) 제공 형태OS형태비트비고WindowsStand-alone32-bit64-bitEclipse Plug-in32-bit1.1.3 Eclipse 지원 참고. 64-bit1.1.3 Eclipse 지원 참고.1.1.2PC 요구 사양WebSquare5 Studio를 설치하고 사용하기 위한 개발자 PC 사양은 아래와 같습니다.표 1-2.PC 설치 사양항목최소권장CPUPentium 4Pentium 4 (2.8Ghz)RAM2GB4GBOSWindows XP/Vista/7/8/10 (32/64-bit)Windows 7/8/10 (32/64-bit)HDD 여유 공간1GB10GB디스플레이1024 * 768 1280 * 10241.1.3Eclipse 지원표 1-3.Eclipse 지원항목내용Eclipse 지원2020-032019-12 (2020/06/12 스튜디오 이후)2019-062019-032018-122018-094.8 (Photon)4.7 (Oxygen)4.6 (Neon)4.5 (Mars)JDK 지원Java 1.8 이상기타CSS3 지원Chromium 64 지원JDK 1.8 지원 Eclipse 4.2 이전 버전의 경우 JDK 1.8을 공식 지원하지 않음.제약 사항(1) Window OS 만 지원.(2) 최소 JRE(JDK) 1.8.0_2xx 설치가 필요.

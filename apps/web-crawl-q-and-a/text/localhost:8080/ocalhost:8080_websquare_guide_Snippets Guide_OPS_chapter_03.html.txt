
Snippets 생성3.Snippets 생성Snippet 으로 생성할 컴포넌트를 선택한 후 마우스 우클릭하여 <Snippet 추가> 메뉴를 선택합니다.그림 3-1.Snippet 추가<Snippet 추가> 메뉴를 선택하면 상세 다이얼로그가 노출됩니다.그림 3-2.Snippets 추가 상세 다이얼로그To directory : 추가할 Snippet 을 저장할 경로Snippet Name : Snippet 이름Description : Snippet 설명Icon : Snippets View 에서 보여질 아이콘Snippets View 에서도 추가한 Snippet 을 확인할 수 있습니다.그림 3-3.snippets_7또한 1.저장한 경로에 2.Snippet 이름의 xml 파일이 생성된 것을 확인할 수 있습니다.그림 3-4.snippets_6Figure 2.상세 다이얼로그의 1.저장 경로는 현재 선택된 Snippets View 의 폴더나 최상위 폴더로 경로로 설정됩니다. 기본 경로는 [Project Workspace]\.settings\[최상위 폴더] 입니다.위 경로로 지정하면 Snippets을 import 하지 않아도 바로 Snippets View 에서 확인할 수 있습니다. 역으로 이외의 경로로 설정하면 <Import Snippets> 메뉴로 생성한 Snippet 파일을 불러와야 Snippets View 에서 보여집니다.


Snippets 소개1.Snippets 소개Snippets 는 사용자가 만들었던 xml 화면 내의 특정 컴포넌트들을 별도의 xml로 저장해 필요할 때 불러서 사용할 수 있습니다. Snippets View 에 만들어 놓은 Snippets (xml 파일) 을 import 하여 사용합니다. 사용자가 별도로 저장한 컴포넌트들의 집합이라고 볼 수 있습니다.컴포넌트로 그림을 그리듯 에디터뷰의 Design View에서 클릭 하나로 화면을 그릴 수 있습니다.그림 1-1.Snippet-View새로운 Project를 생성을 하거나 기존에 사용하고 있는 Project에 Snippets를 Import를 시켰을 경우, Snippets에 사용되는 CSS가 새로 생성된 Project나 기존 Project에 존재해야 하며, 존재하지 않으면 화면이 제대로 표현되지 않습니다. Snippets 는 컴포넌트 마크업만 별도의 xml 파일로 저장하기 때문에 inline-style 만 적용됩니다. 즉 Internal stylesheet, External stylesheet 을 통해 CSS를 사용할 경우 Snippets 를 사용하는 화면에서 CSS 파일을 참조하거나(class 로 CSS를 사용할 때), <head> 의 <style> 태그 안에 CSS 를 직접 정의하여야 합니다.

Event Detail




submit

Submission 요청 전 선처리. return false시 submission 수행 안함.

Parameter

nametypedescription


eObjectSubmission 객체
  e.id            : Submission 객체의 ID 반환





submitdone

Response Status 코드 값이 정상 일 경우 동작.

Parameter

nametypedescription


eObjectSubmission 객체
  e.id                    : Submission 객체의 ID 반환
  e.resourceUri           : 통신 URI 반환
  e.responseHeaders       : Response Headers 내용 반환
  e.responseStatusCode    : Response Status Code 반환
  e.responseReasonPhrase  : Response Status Text 반환
  e.responseBody          : Response Data를 XML 데이터로 parse 한 XML 객체 반환(Response Content-Type이 JSON 인 경우 json객체를 XML로 parse)
  e.responseText          : Response Data 원본으로 String 형태 반환
  e.responseJSON          : Response Data 원본으로 JSON 형태 반환





submiterror

Response Status 코드가 오류 (200 미만 300 이상) 일 경우 동작.

Parameter

nametypedescription


eObjectSubmission 객체
  e.id                    : Submission 객체의 ID 반환
  e.errorType             : target-error 로 고정값 반환
  e.resourceUri           : 통신 URI 반환
  e.responseHeaders       : Response Headers 내용 반환
  e.responseStatusCode    : Response Status Code 반환
  e.responseReasonPhrase  : Response Status Text 반환
  e.responseBody          : Response Data를 XML 데이터로 parse 한 XML 객체 반환(Response Content-Type이 JSON 인 경우 json객체를 XML로 parse)
  e.responseText          : Response Data 원본으로 String 형태 반환
  e.responseJSON          : Response Data 원본으로 JSON 형태 반환

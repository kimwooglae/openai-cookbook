Outline 뷰의 Script 탭은 현재 페이지에 포함된 스크립트 목록을 표시하며, 각 스크립트를 클릭하면 Script 뷰의 상세 코드로 이동할 수 있습니다.그림 9-3.Script 목록9.5Source표 9-4.Source 뷰 설명설명Design 뷰에서 작업한 내용에 대해 자동으로 XML 소스가 생성되어 표시되는 공간.기능개발자가 직접 XML 소스 확인 및 추가 편집이 가능.Workspace 뷰 분리Design 뷰 분리[Window] - [New Editor]를 선택하면 Script 뷰과 Source 뷰만 따로 분리할 수 있습니다. Design 뷰과 Script 혹은 Source 뷰을 동일한 화면에 표시하여 더욱 편리하게 웹 페이지를 디자인할 수 있습니다. Script 혹은 Source 뷰에서 수정한 내용은 CTRL+F5를 통해 Design 뷰에 반영할 수 있습니다.9.6Property10. 기능 및 특징 – 10.10 Property 뷰 참조.9.7Outline10. 기능 및 특징 – 10.11 Outline 뷰 참조.9.8도움말F1 버튼을 구르면 아래와 같이 웹스퀘어 도움말을 확인할 수 있습니다.그림 9-4.도움말그림 9-5.YouTube 동영상 (https://youtu.be/HmwVWyaxZEc)9.9Perspective우측 상단의 Open Perspective 버튼을 클릭하면 화면 Perspective를 설정할 수 있습니다.그림 9-6.Open Perspective그림 9-7.Perspectives그림 9-8.YouTube 동영상 (https://youtu.be/klb9sG73vaU)

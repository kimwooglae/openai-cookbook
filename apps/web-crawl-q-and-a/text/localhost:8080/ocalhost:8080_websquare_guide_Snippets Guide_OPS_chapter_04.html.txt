
Snippets 사용법4.Snippets 사용법Snippet 은 마우스 클릭으로 손쉽게 생성할 수 있습니다.제공되는 Snippets 은 공통 CSS 에서 style 및 위치를 지정하기 때문에, WebSquare 스튜디오의 "Draw Mode" 를 변경해야 합니다.그림 4-1.snippets_12_9위 모드로 설정해야 Snippets 에 설정된 CSS 의 위치가 정상적으로 표현됩니다.
(※ 공통 CSS 에서 위치 설정을 하기 때문에 이 모드를 활성화하시길 바랍니다)화면을 최초로 그릴 때 기본으로 넣는 Layout 을 예시로 들어 Snippet을 그려보겠습니다.먼저, Snippet 을 넣을 위치(컴포넌트)를 선택해주셔야 합니다. 
(※ 미선택시 Body 태그의 자식 노드로 추가됩니다.)이 장에서 넣을 Snippet 은 Layout 용도이기 때문에 Body 바로 자식 노드로 넣습니다.그림 4-2.snippets_12_9_2"change Draw Mode"를 활성화하면 아래의 상세 다이얼로그가 노출됩니다.그림 4-3.생성 상세 다이얼로그기준 객체 : 선택한 컴포넌트 (Component View에서 어떤 컴포넌트도 선택되지 않았을 경우는 Body 태그가 기준이다.)삽입 위치 : 기준 객체를 기준으로 어떤 위치에 삽입할 것인지 선택할 수 있다.float : 해당 컴포넌트에 float 속성을 지정할 수 있으나, 제공된 Snippets 에는 지정하지 않는다(none).표 4-1."삽입 위치" 선택 사항 자식으로 넣을 수 있는 컴포넌트 (block 요소)자식으로 넣을 수 없는 컴포넌트 (inline 요소)1) 기준객체의 마지막 자식 노드2) 기준객체의 첫 번째 자식 노드3) 기준객체 앞에 추가4) 기준객체 뒤에 추가1) 기준객체 앞에 추가2) 기준객체 뒤에 추가

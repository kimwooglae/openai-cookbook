WebSquare.bigDecimal - 5.0_5.4826M.20230213.171915





String 문자열을 받아 bigDecimal 타입으로 반환.
 JavaScript에서 10^22 이상인 정수와 10^(-7) 이하인 소수는 지수 형태로 강제 치환되고 15자리 이하의 소수점은 자동으로 버려짐.  
 이를 보완하기 위해 String 문자열을 받아 bigDecimal 타입으로 반환하는 기능을 제공. 
 WebSquare.bigDecimal.calc는 String 문자열 연산식을 받아 bigDecimal 연산을 수행.
 bigDecimal을 사용하려면 config.xml 파일의 engine 노드에 <module src="externalJS/big/big.min.js"/> 설정해야 함.

calc(
									  str
										)
							

String 문자열 연산식을 받아 bigDecimal 연산을 수행. 단, SP4 이상에서 사용할 경우 $p.bigDecimal.calc() 형태로 사용해야 함.


Parameter

nametyperequireddescription


strStringY문자열 연산식




Return

typedescription


Object연산 결과를 bigDecimal로 반환.


Sample
// (예제1)
var a = $p.bigDecimal("123456789012345678901234567890"); 
var b = $p.bigDecimal("111111111111111111111111111111");
var c = $p.bigDecimal("678901234567890123456789012345");
var d = $p.bigDecimal("345678934567893456789345678934");
var ret = $p.bigDecimal.calc("a + b c + (d - a) / (c + d)"); 
input1.setValue(ret); 
//연산결과가 input1에 올바르게 표현됨.

// (예제2)
var a = $p.bigDecimal("1234");
var b = 3000;
var ret = $p.bigDecimal.calc ("a+b*b-b/(a+b)"); 
// Number + bigDecimal 연산도 가능함. 단 Number는 17자리 이하여야 함.
 
// (예제3)
var a = "abcd"; // NaN
var b = $p.bigDecimal("200");
var ret = $p.bigDecimal.calc("a + b");
input2.setValue(ret); 
// 연산결과는 NaN 데이터를 나타내는 bigDecimal 타입. 숫자 타입이 아닌 데이터와 연산하면 전부 NaN이 반환됨.


compare(
									  data
									       , 
									       type
										)
							

type이 true 인 경우, 입력받은 bigDecimal과 비교하여 더 작은 수를 반환한다.type이 false 이거나 undifined 인 경우, 입력받은 bigDecimal과 비교하여 더 큰 수를 반환한다.


Parameter

nametyperequireddescription


dataObjectY비교 대상 bigDecimal 데이터


typeBooleanN비교 방법 (true - 큰 수 반환, false - 작은 수 반환)




Return

typedescription


Objecttype에 따라 비교한 값들중 더 큰 수, 더 작은 수의 bigDecimal값을 반환한다.


Sample

// 최대값 구하기
var a = WebSquare.bigDecimal("111111111111111111111111111111");
var b = WebSquare.bigDecimal("11111");

var result = a.compare(b); 
var result = a.compare(b,false);
var result = a.compare(b,0);
console.log(result.toString()); //result = WebSquare.bigDecimal("111111111111111111111111111111");

// 최소값 구하기
var a = WebSquare.bigDecimal("111111111111111111111111111111");
var b = WebSquare.bigDecimal("11111");

var result = a.compare(b,true); 
var result = a.compare(b,1);
console.log(result.toString()); //result = WebSquare.bigDecimal("11111");
